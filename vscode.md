# Setup

### Mac

#### To enable `code .` in command line

- Open the **Command Palette** via `Command+Shift+P` and type `shell command`
- Select `Install 'code' command in PATH`

# Shortcuts

## Mac

### Duplicating a line/selection

    Shift + Option + Down

### Multi select

    Command + D

### Multiline select

    Option + Command + arrow-key (up/down)
