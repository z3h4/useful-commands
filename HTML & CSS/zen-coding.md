## Create a list with values

    ul>li*5{Item $}

This will produce the following HTML snippet

    <ul>
      <li>Item 1</li>
      <li>Item 2</li>
      <li>Item 3</li>
      <li>Item 4</li>
      <li>Item 5</li>
    </ul>

## Create 3 headings with values

    h${Heading $}*3

This will produce the following HTML snippet

    <h1>Heading 1</h1>
    <h2>Heading 2</h2>
    <h3>Heading 3</h3>
